Month-wise Goals for Cloud Computing Role

January - February

Goal: Build foundational cloud knowledge and enhance current role-specific skills.

  • Complete Microsoft Certified: Azure Fundamentals (AZ-900).
    Course Link
    How to pursue: Follow Microsoft Learn’s free study path or purchase a Udemy course.
  • Explore basic cloud concepts, compute, storage, and networking on Azure.
  • Practice setting up Azure resources using the Azure portal.

March - April

Goal: Gain hands-on experience with cloud platforms and automation tools.

  • Start Microsoft Certified: Azure Administrator Associate (AZ-104).
    Course Link
    How to pursue: Use official Microsoft resources or paid platforms like A Cloud Guru.
  • Learn Infrastructure-as-Code tools like Terraform.
    Terraform Course Link
    How to pursue: Take HashiCorp’s free learning paths or Udemy courses.
  • Build a project deploying a web application on Azure (e.g., with VMs or App Services).

May - June

Goal: Dive deeper into cloud security and DevOps practices.

  • Complete Microsoft Certified: Azure Security Engineer Associate (AZ-500).
    Course Link
    How to pursue: Enroll in a security-focused Azure training program.
  • Start learning Docker and Kubernetes for containerization.
    Kubernetes for Beginners Link
    How to pursue: Use Udemy or Kubernetes.io documentation.
  • Work on an Azure cloud security-focused project (e.g., setting up Azure Policy or role-based access control).

July - August

Goal: Expand knowledge into multi-cloud environments.

  • Complete AWS Certified Solutions Architect - Associate.
    Course Link
    How to pursue: Use platforms like A Cloud Guru or Linux Academy.
  • Understand basic AWS services like EC2, S3, and IAM.
  • Implement a project comparing similar workloads on Azure and AWS.

September - October

Goal: Gain advanced expertise and focus on specific cloud roles.

  • Start Microsoft Certified: Azure Solutions Architect Expert (AZ-305).
    Course Link
    How to pursue: Follow Microsoft’s learning path and hands-on labs.
  • Dive deeper into Azure DevOps by completing Microsoft DevOps Engineer Expert training (AZ-400).
    AZ-400 Course Link
  • Create a DevOps pipeline on Azure DevOps, integrating CI/CD workflows.

November - December

Goal: Finalize expertise and showcase your skills.

  • Build a comprehensive cloud portfolio, showcasing all completed projects.
  • Apply for cloud-related certifications if not completed earlier.
  • Contribute to open-source cloud projects or participate in hackathons.
  • Prepare for cloud computing interviews by practicing scenario-based questions.
